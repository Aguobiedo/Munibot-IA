# MuniBot Rosario

## Contexto del proyecto

Este proyecto fue desarrollado con fines académicos para la materia **Informática en la Administración Pública** de **Ingeniería en Sistemas de Información** en la **Universidad Tecnológica Nacional - Facultad Regional Rosario (UTN FRRo)**.

> **Importante:** MuniBot Rosario **no está afiliado, avalado ni representa oficialmente** a la Municipalidad de Rosario ni a la Universidad Nacional de Rosario (UNR).

## Descripción

**MuniBot Rosario** es un chatbot web diseñado para ayudar a estudiantes que llegan a vivir a la ciudad de Rosario. Centraliza información sobre alojamiento, barrios, servicios y actividades deportivas, combinando respuestas predefinidas con un módulo de inteligencia artificial conectado a Internet para ofrecer información actualizada.

El sistema busca simplificar la búsqueda de recursos esenciales mediante una interfaz intuitiva y de fácil navegación, proporcionando enlaces a fuentes oficiales y datos relevantes para facilitar la adaptación de los nuevos estudiantes a la vida en la ciudad. Además, fue desarrollado con una arquitectura modular que permite incorporar nuevas categorías y funcionalidades de forma sencilla en futuras versiones.
